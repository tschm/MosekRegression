mosek_tools.solver.__l1_norm(model, name, expr)[source]

Given an expression (e.g. a vector) this returns the L1-norm of this vector as an expression. It also introduces n (where n is the size of the expression) auxiliary variables. Mosek requires a name for any variable that is added to a model. The user has to specify this name explicitly. This requirement may disappear in future version of this API.

ATTENTION: THIS WORKS ONLY IF expr is a VARIABLE

mosek_tools.solver.__l2_norm(model, name, expr)[source]

Given an expression (e.g. a vector) this returns the L2-norm of this vector as an expression. It also introduces an auxiliary variables. Mosek requires a name for any variable that is added to a model. The user has to specify this name explicitly. This requirement may disappear in future version of this API.

Given an expression (e.g. a vector) this returns the squared L2-norm of this vector as an expression. It also introduces an auxiliary variables.
